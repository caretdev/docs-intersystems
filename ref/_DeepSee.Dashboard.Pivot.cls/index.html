%DeepSee.Dashboard.Pivot

Class %DeepSee.Dashboard.Pivot Extends %DeepSee.UserLibrary.FolderItem [ System = 4 ]

This persistent class represents a saved DeepSee pivot table.

Parameter XMLNAME = "pivot";
