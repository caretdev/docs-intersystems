%DeepSee.Time.Partial.WeekNumber

Class %DeepSee.Time.Partial.WeekNumber Extends %DeepSee.Time.AbstractLevel [ System = 3 ]

This class implements a ISO 8601 week number level within a time dimension. This represents the Week number (1 through 53) independent of year.

Parameters

CALENDAR

Parameter CALENDAR As STRING = "partial";

%Convert

ClassMethod %Convert(pTime As %DeepSee.Datatype.dateTime, pTimeOffset As %String = "") As %Integer

Convert a partial date ODBC value to the logical value used for this level. This is used within the computed field logic for properties within a fact table based on this level.
. In this case, we convert a prtial date to a week number: That is convert to a number 1 through 53. If the partial date is not complete, it will be converted to the "0" week number for Unknown.
