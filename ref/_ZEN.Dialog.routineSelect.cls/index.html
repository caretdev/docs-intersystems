%ZEN.Dialog.routineSelect

Class %ZEN.Dialog.routineSelect Extends standardDialog [ Deprecated, System = 4 ]

This utility dialog window displays and lets the user select from a set of routines.
The user must have Use privileges for the %Development resource in order to use this page. If Namespace is passed in then switch to that namespace to run the query
